# Event Booking System

A full-stack event booking platform where users can browse events, reserve seats, and manage their bookings. Built as a monorepo with a NestJS backend handling bookings, queued background jobs, and persistence, and a React frontend for the user experience.

## Stack

- **Backend:** NestJS, PostgreSQL, Redis (BullMQ)
- **Frontend:** React, TypeScript
- **Infrastructure:** Docker Compose (PostgreSQL + Redis)
